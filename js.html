¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web, como animaciones, validaciones de formularios, etc.

What is JavaScript?

JavaScript is a programming language that allows you to add interactivity to web pages, such as animations, form validations, etc.

Temas clave / Key Topics

  • Variables: let, const, var
  • Funciones / Functions: tradicionales y flecha
  • Eventos / Events: onclick, oninput, etc.
  • DOM: manipulación de elementos HTML
  • Condicionales y bucles / Conditionals & loops: if, for, while

Ejemplo básico / Basic Example

document.querySelector("h1").addEventListener("click", () => {
  alert("¡Hola desde JavaScript!");
});
