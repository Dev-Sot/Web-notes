¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para añadir interactividad y dinamismo a las páginas web. Funciona en el navegador del usuario y puede manipular el contenido, el estilo y la estructura de un documento HTML.

What is JavaScript?

JavaScript is a programming language used to add interactivity and dynamism to web pages. It runs in the user's browser and can manipulate the content, style, and structure of an HTML document.

Temas clave / Key Topics

  • Variables: let (variable de bloque), const (constante), var (antigua, evita usarla).
  • Tipos de datos: string, number, boolean, object, array, null, undefined.
  • Funciones:
    • Tradicionales: function nombre() { ... }
    • Flecha: const suma = (a, b) => a + b;
  • Eventos: onclick, oninput, addEventListener.
  • DOM: seleccionar y modificar elementos HTML con document.querySelector, innerHTML, style, etc.
  • Condicionales: if, else, switch.
  • Bucles: for, while, forEach.

Ejemplo básico / Basic Example

// Mostrar un mensaje al hacer clic en el título
document.querySelector("h1").addEventListener("click", () => {
  alert("¡Hola desde JavaScript!");
});

Ejemplo de Variables y Operaciones

// Declarar variables
let nombre = "Nelson";
const edad = 20;
let esEstudiante = true;

// Operación básica
let añoNacimiento = 2025 - edad;

console.log("Nombre:", nombre);
console.log("Año de nacimiento:", añoNacimiento);
console.log("¿Es estudiante?", esEstudiante);

Ejemplo de Condicionales

let temperatura = 30;

if (temperatura > 25) {
  console.log("Hace calor");
} else {
  console.log("El clima está agradable");
}

Ejemplo de Bucle

let frutas = ["Manzana", "Banana", "Naranja"];

for (let i = 0; i < frutas.length; i++) {
  console.log("Fruta:", frutas[i]);
}

Ejemplo de Interacción con el DOM

// Cambiar el color de un párrafo con id="demo"
document.getElementById("demo").style.color = "red";

// Cambiar el texto
document.getElementById("demo").innerHTML = "Texto cambiado por JavaScript";
