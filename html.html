¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear páginas web. Permite estructurar el contenido mediante etiquetas, que definen títulos, párrafos, imágenes, enlaces, listas, tablas, formularios y otros elementos. No es un lenguaje de programación, sino de marcado, lo que significa que describe la estructura y el significado del contenido.

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It allows you to structure content using tags that define headings, paragraphs, images, links, lists, tables, forms, and more. It is not a programming language but a markup language that describes the structure and meaning of content.

Etiquetas HTML principales

Categoría Etiquetas Descripción
Etiquetas básicas <p>, <h1>-<h6>, <a>, <img> Se usan para párrafos, títulos, enlaces e imágenes.
Listas <ol>, <ul>, <li> Listas ordenadas y desordenadas para organizar información.
Tablas <table>, <tr>, <td>, <th> Permiten mostrar datos en filas y columnas.
Formularios <form>, <input>, <label>, <button> Capturan datos del usuario como texto, opciones o botones.
Semántica <header>, <main>, <footer> Mejoran la estructura y comprensión del contenido.

2. Listas

<ol>
  <li>Elemento uno</li>
  <li>Elemento dos</li>
  <li>Elemento tres</li>
</ol>

<ul>
  <li>Elemento A</li>
  <li>Elemento B</li>
  <li>Elemento C</li>
</ul>

Resultado:

  1. Elemento uno
  2. Elemento dos
  3. Elemento tres
  • Elemento A
  • Elemento B
  • Elemento C
