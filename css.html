¿Qué es CSS?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es el lenguaje que se utiliza para describir la apariencia y el formato de un documento HTML.

Permite separar el contenido de la presentación, mejorando la organización y el mantenimiento del código.

Tipos de CSS

  • En línea: dentro de una etiqueta HTML con el atributo style.
  • Interno: dentro de una etiqueta <style> en el <head>.
  • Externo: en un archivo separado con extensión .css enlazado con <link>.
<link rel="stylesheet" href="estilos.css">

Selectores

  • p → Todos los párrafos.
  • .clase → Elementos con esa clase.
  • #id → Elemento con ese identificador.
  • elemento elemento → Elementos dentro de otros.
  • elemento:hover → Estado al pasar el mouse.
p {
  color: blue;
}
#titulo {
  font-size: 24px;
}
.boton:hover {
  background-color: yellow;
}

Propiedades comunes

  • color → color del texto.
  • background-color → color de fondo.
  • margin → espacio exterior.
  • padding → espacio interior.
  • border → borde del elemento.
  • font-family → tipo de letra.
body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
  color: #333;
  margin: 0;
  padding: 0;
}

Modelo de Caja (Box Model)

Todos los elementos en HTML se representan como cajas que incluyen:

  • Content: el contenido.
  • Padding: espacio entre el contenido y el borde.
  • Border: el borde del elemento.
  • Margin: espacio exterior al borde.
div {
  margin: 20px;
  padding: 10px;
  border: 2px solid black;
}

Diseño con Flexbox

.container {
  display: flex;
  justify-content: center; /* alinea horizontalmente */
  align-items: center; /* alinea verticalmente */
}

Diseño con Grid

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

Media Queries

Permiten adaptar el diseño según el tamaño de la pantalla.

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
