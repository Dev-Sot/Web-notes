¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) es el lenguaje que se usa para dar estilo a los documentos HTML. Permite controlar colores, fuentes, márgenes y más.

What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML documents. It allows you to control colors, fonts, margins, and more.

Temas clave / Key Topics

  • Selectores / Selectors: elemento, .clase, #id
  • Propiedades / Properties: color, background, margin, padding
  • Modelo de caja / Box Model
  • Flexbox y Grid para layouts
  • Media Queries para diseño responsivo

Ejemplo básico / Basic Example

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
  color: #333;
}
